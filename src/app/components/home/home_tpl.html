Directiva de control de entrada de datos numéricos para montos

Esta es una directiva de atributo que permite gestionar la entrada de datos numéricos para montos. Se controlará su validez, formato y cantidad de decimales. El número se va ir formateando a medida que se ingrese el valor

Se debe vincular el dato de entrada a un elemento del modelo, por tanto es obligatorio el atributo ng-model

El input tendrá las siguientes validaciones:
  • Solo se permite ingresar números positivos
  • En caso de asignar un número de decimal mayor a 0, al ingresar un "." lo reemplazará por una ","
  • Valor mínimo: Opcionalmente se puede agregar un atributo format-number-min con el valor mínimo permitido.
  • Valor máximo: Opcionalmente se puede agregar un atributo format-number-max con el valor máximo permitido.

Uso

Para ser utilizada es necesario agregar 2 atributos a los elementos

  • format-number
  • ng-model

format-number

Llama a la directiva, funciona sobre elementos inputs.

ng-model

Indica la propiedad del modelo que está enlazada con el campo de entrada de datos.

Dependencias

Agregar dependencia del módulo WebBancoChile.formatNumDirective

Uso

Como se explica anteriormente esta directiva tiene como objetivo gestionar la entrada de datos numéricos, la forma de utilizarla es agregando atributos a un elemento como se detalla a continuación:

<input format-number="{{vm.nroDecimales}}" ng-model="vm.miNumero" format-number-min="{{vm.numeroMin}}" format-number-max="{{vm.numeroMax}}" >

Esto está indicando que el número sea formateado con 3 decimales, con un valor mínimo a 10.000 y un valor mayor a 30.000

Ejemplo

El número no puede ser menor a {{vm.numeroMin}} El número no puede ser mayor a {{vm.numeroMax}}
